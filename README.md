# ExDuck

An Elixir Client for the DuckDuckGo Instant Answer API

## Installation

ExDuck can be installed by adding `ex_duck` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:ex_duck, "~> 0.1.0"}
  ]
end
```

Documentation is [published on hexdocs](https://hexdocs.pm/ex_duck).
